## System Overview

* os: `Arch`
* wm: `i3`
* bar: `polybar`
* dm: `lightDM`
* compositor: `picom`
* shell: `zsh`
* audio: `pipewire`
* vpn: `openconnect`
* editor: `neovim`
* terminal: `alacritty`
* terminal font: `Source Code Pro`
* file browser: `thunar`
* notifications: `dunst`
* browser: `firefox`
* pdf reader: `zathura`
* wallpaper: `feh`
* notes: `obsidian`
