# dotfiles

My dotfiles managed via a git bare repo.

## System Overview

* os: `Arch`
* wm: `i3`
* dm: `lightDM`
* compositor: `picom`
* shell: `zsh`
* audio: `pipewire`
* editor: `neovim`
* terminal: `alacritty`
* terminal font: `Source Code Pro`
* file browser: `thunar`
* notifications: `dunst`
* browser: `firefox`
* pdf reader: `zathura`
* wallpaper: `feh`
