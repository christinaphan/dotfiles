# dotfiles

My dotfiles managed via a git bare repo.

## System Overview

* os: `Arch`
* wm: `i3`
* compositor: `picom`
* shell: `zsh`
* editor: `neovim`
* terminal: `alacritty`
* terminal font: `Source Code Pro (Sauce Code Pro for Nerd Font)`
* file browser: `thunar`
* notifications: `dunst`
* browser: `firefox`
